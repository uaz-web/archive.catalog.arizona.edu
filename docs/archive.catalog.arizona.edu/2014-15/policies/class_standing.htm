Class Standing/Classification Policy

Undergraduates need to earn the following minimum number of units to attain the next higher classification:

  • 0 units for Freshman standing
  • 30 units for Sophomore standing
  • 60 units for Junior standing
  • 90 units for Senior standing

A studentï¿½s class standing does not necessarily correspond to the number of semesters or units required to complete all degree requirements, as some degree programs require more than 120 units. See the Academic Advisement Reports for the minimum number of units needed for each UA undergraduate degree program.
